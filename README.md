# Comments

A simple app for adding comments

## Start

1. Clone [repo](https://github.com/OwtmPatrick/comments-react-mobx-ts)
2. Install packages

```
npm install
```

3. Start dev server

```
npm run start
```

## Build

Run following command to build your source code:

```
npm run build
```
